# Flight-Fare-Project

## Project Overview

This project aims to predict airline ticket prices based on various factors such as airline, source, destination, journey date, departure time, arrival time, duration, and total stops. Machine Learning techniques are used to build a predictive model that estimates flight fares accurately.

## Problem Statement

The objective of this project is to analyze flight data and develop a machine learning model that can predict flight ticket prices.

## Results

The Random Forest Regressor achieved the best performance and was selected as the final model for flight fare prediction.
